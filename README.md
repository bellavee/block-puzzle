# PyPuzzle

A block puzzle game write in Python and PyGame.

![1-ani](https://user-images.githubusercontent.com/74190308/152658977-cff04a77-f01f-42bd-91db-bac83f42db93.PNG)

## Require
Python3 and PyGame

Install PyGame
``pip3 install pygame``

## How to start
Using the command line
``python3 pypuzzle.py``
